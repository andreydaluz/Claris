# Placeholders de imagens

Substitua estes arquivos mantendo os mesmos nomes para trocar rapidamente o visual da landing page:

- `hero-background-placeholder.svg` → imagem de fundo da hero
- `hero-house-placeholder.svg` → imagem principal da casa na hero
- `smoke-overlay-placeholder.svg` → camada de fumaça/névoa da hero
- `product-placeholder.svg` → imagem da seção de produto
- `floorplan-3suites-placeholder.svg` → planta 3 suítes
- `floorplan-4suites-placeholder.svg` → planta 4 suítes
- `masterplan-placeholder.svg` → implantação do condomínio

Também é possível ajustar os caminhos diretamente no `index.html` e trocar textos de cada planta nos blocos da seção `#plantas`.
